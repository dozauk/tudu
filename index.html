Google Drive

Connect Google Drive to sync your data across devices, attach files to tasks and events, and share planners or projects with others.

How to get a Client ID
  1. Go to console.cloud.google.com
  2. Create a project (or select an existing one)
  3. Enable the Google Drive API under APIs & Services → Library
  4. Go to APIs & Services → Credentials → Create Credentials → OAuth 2.0 Client ID
  5. Application type: Web application
  6. Authorised JavaScript origins — add both:
    https://www.doza.org
    http://localhost:8080
  7. Copy the Client ID and paste it above
